# 03 — Strings

> Strings are JavaScript's immutable, array-like sequence of characters. Master what immutability really means for performance, the canonical string patterns — **two pointers, sliding window, frequency map, expand-around-center** — and the built-in methods that let you work with text at speed.

## Learn Path

1. **Read [`notes.md`](./notes.md) first** — concepts, immutability, JS internals, complexity table, patterns, gotchas, templates.
2. Solve questions in order. Each lives in `questions/<NN>-<slug>.md`.
3. Write your solution in the matching `solutions/<NN>-<slug>.js`.
4. Tick off the checklist below as you finish.
