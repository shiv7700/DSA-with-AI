(easy → medium → hard). Solve them in a `solutions/` subfolder as you go.

## How to Use

For each topic:
1. Read the **Concept Check** and answer in your own words.
2. Implement any "Implement First" data structure from scratch.
3. Work through Easy → Medium → Hard.
4. Save solutions like:
   ```
   02-Arrays/
     questions.md
     solutions/
       01-reverse-array.js
       02-min-max.js
   ```
5. Add a one-line note at the top of each solution: time/space complexity.

## Suggested Study Order

- **Week 1–2:** Foundations (1–6)
- **Week 3–4:** Linear DS (7–10)
- **Week 5:** Two Pointers + Sliding Window (19, 20) — apply on what you know
- **Week 6–7:** Trees, BST, Heaps, Tries (11–14)
- **Week 8–9:** Graphs + DSU (15, 16)
- **Week 10:** Greedy + Backtracking (23, 24)
- **Week 11:** Divide & Conquer (25)
- **Week 12+:** Dynamic Programming (26) — the longest topic, do it last and slow
- **Bonus:** Bit Manipulation, Math, Segment / Fenwick Trees as needed
