# 23 — Greedy Algorithms

> Make the locally-best choice at each step and trust that the choices add up to a globally-optimal answer. Sometimes this works beautifully. Sometimes it doesn't. The art is knowing which is which — and being able to prove it.

## Learn Path

1. **Read [`notes.md`](./notes.md) first** — the greedy mindset, when it works, when it fails, exchange arguments, interval scheduling, heap patterns, interview recognition.
2. Solve questions in order. Each lives in `questions/<NN>-<slug>.md`.
3. Write your solution in the matching `solutions/<NN>-<slug>.js`.
4. Tick off the checklist below as you finish.
