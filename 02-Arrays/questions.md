# Arrays

> Master the workhorse data structure of JavaScript.

## Concept Check

1. Are JavaScript arrays true arrays or objects? What does V8 do under the hood?
2. Difference between `splice`, `slice`, and `split`.
3. Complexity of: `push`, `pop`, `shift`, `unshift`, `indexOf`, `includes`.
4. `map` vs `forEach` vs `reduce` — when do you use each?
5. Shallow copy vs deep copy of an array — show 3 ways to shallow copy.
6. What is a sparse array? How does it differ in iteration?

## Easy

1. Reverse an array **in place** (no `.reverse()`).
2. Find the maximum and minimum in an array in a single pass.
3. Find the sum and average of an array.
4. Remove duplicates from an array (preserve order).
5. Check if an array is sorted (ascending).
6. Rotate an array to the right by `k` positions.
7. Merge two sorted arrays into one sorted array.
8. Move all zeros to the end (preserve order of non-zeros).
9. Find the second largest number in an array.
10. Given an array, return an array of squares sorted ascending.

## Medium

11. **Two Sum** — return indices of the two numbers that add up to `target`. O(n).
12. **Best Time to Buy & Sell Stock** — single transaction, maximize profit.
13. **Product of Array Except Self** — without using division, O(n).
14. **Maximum Subarray (Kadane's)** — largest contiguous sum.
15. **Container With Most Water** — two-pointer classic.
16. **3Sum** — all unique triplets that sum to zero.
17. **Find All Duplicates** in an array where 1 ≤ a[i] ≤ n, O(n) time, O(1) extra space.
18. **Sort Colors** (Dutch National Flag) — sort 0s, 1s, 2s in one pass.
19. **Subarray Sum Equals K** — count subarrays with sum = k.
20. **Spiral Matrix** — traverse a 2D matrix in spiral order.
21. **Rotate 2D Matrix by 90°** in place.
22. **Set Matrix Zeroes** — if an element is 0, set its entire row and column to 0, O(1) space.

## Hard

23. **Trapping Rain Water** — compute trapped water given elevation map.
24. **First Missing Positive** — O(n) time, O(1) extra space.
25. **Median of Two Sorted Arrays** — O(log(min(m, n))).
26. **Sliding Window Maximum** — max in every window of size `k`.
27. **Merge k Sorted Arrays**.
28. **Longest Consecutive Sequence** — O(n).
