# 02 — Arrays

> Arrays are JS's workhorse linear data structure. Master how V8 actually stores them, the hidden complexity of common methods, and the canonical patterns: **two pointers, sliding window, prefix sum, Kadane's**, hash-map lookup.

## Learn Path

1. **Read [`notes.md`](./notes.md) first** — concepts, JS internals, complexity table, patterns, gotchas, templates.
2. Solve questions in order. Each lives in `questions/<NN>-<slug>.md`.
3. Write your solution in the matching `solutions/<NN>-<slug>.js`.
4. Tick off the checklist below as you finish.
