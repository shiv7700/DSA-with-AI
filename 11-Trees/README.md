# 11 — Trees (Binary Trees)

> Trees are recursion's natural home. Master the four traversals and most tree problems become a clean variation of "recurse left, recurse right, combine with the current node." Get the template into muscle memory and a whole class of hard-looking problems becomes approachable.

## Learn Path

1. **Read [`notes.md`](./notes.md) first** — tree vocabulary, types of binary trees, all four traversals walked through step by step, the universal recursive template, tree DP, and a catalogue of problem shapes.
2. Solve questions in order. Each lives in `questions/<NN>-<slug>.md`.
3. Write your solution in the matching `solutions/<NN>-<slug>.js`.
4. Tick off the checklist below as you finish.
