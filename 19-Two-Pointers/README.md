# 19 — Two Pointers

> Two Pointers is a **pattern**, not a data structure. When your input is sorted, symmetric, or sequential, two pointers usually drops you from O(n²) to O(n) by walking two markers through the data in a coordinated way — so you never look at the same pair twice.

## Learn Path

1. **Read [`notes.md`](./notes.md) first** — the three flavors, walk-throughs with diagrams, Floyd's cycle detection, why this is O(n), templates.
2. Solve questions in order. Each lives in `questions/<NN>-<slug>.md`.
3. Write your solution in the matching `solutions/<NN>-<slug>.js`.
4. Tick off the checklist below as you finish.
