# 05 — Searching

> Linear search is the honest brute force. Binary search is the clever half-and-half. Mastering binary search means mastering the idea of **eliminating half the space on every step** — and knowing exactly where the edges are.

## Learn Path

1. **Read [`notes.md`](./notes.md) first** — concepts, ASCII diagrams, edge cases, common traps, patterns.
2. Solve questions in order. Each lives in `questions/<NN>-<slug>.md`.
3. Write your solution in the matching `solutions/<NN>-<slug>.js`.
4. Tick off the checklist below as you finish.
